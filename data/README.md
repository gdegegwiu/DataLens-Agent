# CSV Datasets

These small CSV files can be used to compare DataLens Agent behavior across different dataset shapes.

| File | Rows | Columns | Use case |
| --- | ---: | ---: | --- |
| `sample_sales.csv` | 20 | 7 | Built-in business sales sample for the default demo. |
| `iris.csv` | 150 | 5 | Classic flower measurement dataset with numeric measurements and one species category. |
| `tips.csv` | 244 | 7 | Restaurant bill and tip dataset with numeric and categorical fields. |

Sources:

- `iris.csv`: https://raw.githubusercontent.com/mwaskom/seaborn-data/master/iris.csv
- `tips.csv`: https://raw.githubusercontent.com/mwaskom/seaborn-data/master/tips.csv
